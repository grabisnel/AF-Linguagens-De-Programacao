## Descrição

Site completo de uma livraria fictícia chamada **Reverie**, com três páginas: Home, Galeria e Contato. Desenvolvido com HTML5 semântico, CSS externo, Bootstrap 5 e JavaScript puro.

## Tecnologias Usadas

- HTML5
- CSS3 (Flexbox, variáveis CSS)
- Bootstrap 5
- JavaScript (DOM)
- Google Fonts (Merriweather + Inter)

## Páginas

- `index.html` — Home com carrossel, cards de destaque e chamadas para ação
- `galeria.html` — Galeria de livros
- `contato.html` — Formulário de contato, mapa e informações da loja
